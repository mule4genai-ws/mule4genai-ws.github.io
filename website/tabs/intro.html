MuleSoft for Salesforce Data Cloud Workshop

Introduction

This workshop is designed to showcase how MuleSoft complements Salesforce Data Cloud capabilities specifically relating to data ingestion, transformation, and activation.

The focus of this workshop is to highlight the MuleSoft technical capabilities within the context of Salesforce Data Cloud i.e., technical use cases where MuleSoft adds value to Data Cloud. This workshop is designed to complement the Salesforce Data Cloud workshop and will not deep dive into Salesforce Data Cloud capabilities.

Using the Watch & Learn format this workshop will deliver majority of the content through detailed product demo walk-through and a segment of the workshop will provide the participants with hands-on experience of the product capability.

The target audience for this workshop is primarily Salesforce Data Cloud customers and prospects interested in understanding how MuleSoft can assist their Data Cloud journey.
