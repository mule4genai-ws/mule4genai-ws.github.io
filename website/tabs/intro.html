MuleSoft for Agentforce / Gen AI Workshop

Introduction

This Watch & Learn workshop will guide you through technical use cases, demonstrating how MuleSoft supports Gen AI integration. You'll gain hands-on insights and detailed product walkthroughs, allowing you to observe, learn, and get started effectively.

This workshop will cover:

  • Module 1: Using Agentforce with MuleSoft
    Including a demo of Agentforce using Topics and Actions to answer questions and perform actions with 3rd party systems using Mulesoft.
  • Module 2: Interacting with Large Language Models (LLMs)
    Walk-through of building a System API using MuleSoft AI Chain Connectors, hands-on viewing and invoking of the System API from the Anypoint Platform.
  • Module 3: Securing LLM Interactions
    Walk-through of applying policies using MuleSoft's API Manager to secure LLM System API, including hands-on testing with different input types.
  • Module 4: LLM-powered Automation
    Walk-through of building workflows powered by LLMs, including retrieval-augmented generation (RAG), with hands-on testing using the UI or API.
  • Wrap-up
    Recap and resources to continue the AI journey with MuleSoft.
