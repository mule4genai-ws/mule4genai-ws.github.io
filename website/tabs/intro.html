MuleSoft for Agentforce / Gen AI Workshop

Introduction

Welcome to the MuleSoft for Salesforce Gen AI Workshop. This workshop aims to introduce you to the capabilities of MuleSoft in integrating and enhancing Salesforce Gen AI functionalities. Learn how to harness the power of AI, connect various data sources, and create intelligent workflows that enrich customer experiences.

This workshop will guide you through technical use cases and demonstrate how MuleSoft can support Gen AI integration, providing hands-on experience and detailed product walk-throughs to help you get started effectively.

This workshop will cover:

  • Module 1: Using Agentforce with MuleSoft
    Including a demo with Topic Center and hands-on configuration of Topics in Anypoint platform.
  • Module 2: Interacting with Large Language Models (LLMs)
    Walk-through of building LLM System API using MAC Connector, with hands-on viewing and invoking of LLM System API from mocking service/console.
  • Module 3: Securing LLM Interactions
    Walk-through of applying policies using LLM Gateway to secure LLM System API, including hands-on testing with different input types.
  • Module 4: LLM-powered Automation
    Walk-through of Agent building concepts, including retrieval-augmented generation (RAG), with hands-on testing using the UI or API.
  • Wrap-up
    Recap and resources to continue the AI journey with MuleSoft.
